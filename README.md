# Sistema Imaginatics V2 - Laravel

Sistema de gestión de clientes, servicios y pagos desarrollado en Laravel 12.

## Características
- ✅ Autenticación
- ✅ Gestión de Clientes
- ✅ Servicios Contratados
- ✅ Consulta RUC (Factiliza API)
- ✅ Pagos Pendientes
- ✅ Registro de Pagos Multi-servicio
